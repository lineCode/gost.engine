# GoST
2D/3D Game Engine based on plugins for C++11

![Imgur](https://i.imgur.com/GjvRTaI.jpg)
![Imgur](https://i.imgur.com/PeOxnAs.jpg)

## Features
* D3D11 render plugin
* .obj model import
* .png .bmp image import
* 2D sprite with animation
* audio loading/audio streaming (.wav .ogg)
* XML read/write
